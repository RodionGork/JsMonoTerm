#JsMonoTerm

Javascript monochrome terminal for rendering in web-pages.

Supported methods:

- print('some text')
- println('some text')
- move(x, y)
- clear()
- addLine(n)
- delLine(n)

Method "getc" if not null is called after user types a character (and is
reset to null).

Example "guesser.html" is provided.
